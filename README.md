## üìÅ Dataset Structure
The dataset consists of 9 files linked by unique identifiers:
* olist_customers_dataset.csv: Customer location data.
* olist_orders_dataset.csv: Core table with order timestamps.
* olist_order_items_dataset.csv: Price and freight information.
* olist_products_dataset.csv: Product categories and dimensions.
